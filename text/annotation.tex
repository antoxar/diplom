The   degree   project   contains of \pageref{LastPage}  pages,  \total{figurecnt} pictures,  \total{tablecnt}  tables,  \bibnum~references  and   \total{sect} appendixs.
The objects of designing are the materials for training students to perform laboratory work on "The discovery of vulnerabilities and remedies for the dynamic analysis of executable files”. Develop techniques and variations of these labs. 
The result is a methodology and software package for teaching students the basics of reverse engineering. Programs for laboratory work were developed in programming environment Visual Studio 2005. The developed programs are a visual aid for the novice researcher executable files, and are listed in the draft vulnerabilities represent a benefit for secure programming.
The safety and ecological compatibility analysis is executed. The feasibility report of the developed system is realized.
