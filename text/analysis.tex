%\addcontentsline{toc}{section}{Анализ технического задания}
\Analysis

\par Согласно техническому заданию в проекте требуется разработать курс лабораторных работ по статическому изучению исполняемых бинарных фалов. Привести теоретическое описание всех существующих на данный момент типов уязвимостей, методов эксплуатации и обнаружения уязвимостей, а также методов защиты от отладки. Для обучения студентов необходимо разработать методику выполнения лабораторных работ, а также требуется создать программный комплекс для практических занятий. 
\par Для разработки такого рода программного комплекса требуется провести аналитический обзор существующих уязвимостей и методов антиотладки, выявить их основные особенности.  При разработке необходимо учитывать специфику современных ОС. Программная реализация осуществляется в среде программирования Visual Studio 10.
